¿QUÉ ES BOOTSTRAP?

Bootsrap es un framework que nos permite utilizar una serie de clases ya definidas de CSS para que nos ahorremos tiempo en definir los selectores y podamos ser más eficientes a la hora de maquetar y estilar la página web al mismo tiempo
