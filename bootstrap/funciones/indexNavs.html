Navbar


Los navbar son un elemento muy importante a la hora de navegar, para que el usuario no se pierda en ningún momento de la página web. Al utilizar Bootstrap, los Navbar se pueden hacer de manera sencilla y profesional. En este artículo veremos cómo hacer algunos ejemplos.

Los componentes de un navbar se pueden dividir en:

Logo: Es el logo o imagen de la página web que pondremos en la navbar. Para ello utilizaremos la clase navbar-brand. Da igual la imagen que tengamos, con que controlemos el tamaño horizontalmente Bootstrap controlará el tamaño verticalmente.

Activo: Este es un tipo de link que le cambia el estilo y lo deja brillando constantemente. Para poner este código tendríamos que poner lo siguiente: nav-link active.

disabled: Este es un tipo de link que le cambia el estilo y lo deja deshabilitado constantemente. Para poner este código tendríamos que poner lo siguiente: nav-link disabled.

Normal: Este es un tipo de link que le cambia el estilo dependiendo si pasas encima de el o no. Para poner este código tendríamos que poner lo siguiente: nav-link.
