Form validation

A la hora de validar un formulario, es común validar los datos antes y después de que el servidor empiece a procesar la información.

Para ello tenemos que comprobar si el usuario está introduciendo la información correcta o si está rellenando la información necesaria para que el servidor pueda procesar todos los datos de manera correcta.

Para poder darle este tipo de efectos, simplemente en el formulario tenemos que poner la clase was-validated y te dará el formato.

Si quieres poner mensajes personalizados, tienes que poner un div con clases valid-feedback e invalid-feedback.
